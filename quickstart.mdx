## Setup Process

### 1. Create an Organization

First, create your organization in our [dashboard](https://dashboard.chipipay.com/).

### 2. Configure Auth Provider

Next, head to [dashboard configuration](https://dashboard.chipipay.com/configure) and add your JWKS Endpoint from your Auth Provider. 

We use this to verify the Bearer Tokens from your Auth Provider used in your frontend.

After adding the JWKS Endpoint, you'll receive a Public Key (`pk_prod_xxxx`). Keep this handy as you'll need it to initialize the SDK.

<Tip>
  You can also refer to one of our sample apps if you want to
  clone a full application rather than embed into an existing one!
</Tip>

## Select your Framework

<Tabs>
  <Tab title="Next.js">
    ## Install the SDK

    ```bash
    npm install @chipi/chipi-sdk
    # or
    yarn add @chipi/chipi-sdk
    # or
    pnpm add @chipi/chipi-sdk
    ```

    ## Initialize the SDK

    ```tsx
    // app/providers.tsx
    'use client'

    import { ChipiProvider } from '@chipi-pay/chipi-sdk'

    export function Providers({ children }: { children: React.ReactNode }) {
      return (
        <ChipiProvider
          apiKey={process.env.NEXT_PUBLIC_CHIPI_API_KEY}
        >
          {children}
        </ChipiProvider>
      )
    }
    ```

    ```tsx
    // app/layout.tsx
    import { Providers } from './providers'

    export default function RootLayout({
      children,
    }: {
      children: React.ReactNode
    }) {
      return (
        <html lang="en">
          <body>
            <Providers>{children}</Providers>
          </body>
        </html>
      )
    }
    ```
