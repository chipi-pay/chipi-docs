# Setup Process

## 1. Setup the Gasless in the Dashboard

First, create your organization in our [dashboard](https://dashboard.chipipay.com/).

Next, head to [Gasless Settings](https://dashboard.chipipay.com/configure/gasless)

You'll need to add a JWKS Endpoint from your Auth Provider to authenticate the sponsoring of transactions.

<div className="flex justify-center">
  <img src="/images/gasless/configure-gasless-1.png" alt="JWKS Setup" />
</div>

Which will give you access to your Public Key (`pk_prod_xxxx`). Keep this handy as you'll need it to initialize the SDK.

<div className="flex justify-center">
  <img src="/images/gasless/configure-gasless-2.png" alt="JWKS Setup" />
</div>

You won't need the Secret Key (`sk_prod_xxxx`) for the SDK.
As all requests are validated by the JWKS Endpoint using your auth provider.

<Tip>
  You can also refer to one of our sample apps if you want to clone a full
  application rather than embed into an existing one!
</Tip>

## 2. Setup the Chipi SDK in your project to use the Gasless

### Select your Framework

<Tabs>
  <Tab title="Next.js">
    ## Install the SDK

    ```bash
    npm install @chipi/chipi-sdk
    # or
    yarn add @chipi/chipi-sdk
    # or
    pnpm add @chipi/chipi-sdk
    ```

    ## Initialize the SDK

    ```tsx
    // app/providers.tsx
    'use client'

    import { ChipiProvider } from '@chipi-pay/chipi-sdk'

    export function Providers({ children }: { children: React.ReactNode }) {
      return (
       <ChipiProvider
        config={{
          apiPublicKey: process.env.NEXT_PUBLIC_CHIPI_API_KEY!,
          secretKey: process.env.NEXT_PUBLIC_CHIPI_SECRET_KEY!,
        }}
      >
          {children}
        </ChipiProvider>
      )
    }
    ```

    ```tsx
    // app/layout.tsx
    import { Providers } from './providers'

    export default function RootLayout({
      children,
    }: {
      children: React.ReactNode
    }) {
      return (
        <html lang="en">
          <body>
            <Providers>{children}</Providers>
          </body>
        </html>
      )
    }
    ```

  </Tab>
