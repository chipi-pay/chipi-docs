## Welcome

The ChipiPay API is a RESTful API that allows you to interact with the ChipiPay platform.
## Authentication

All API endpoints are authenticated using Bearer tokens and picked up from the specification file.

```json
"security": [
  {
    "bearerAuth": []
  }
]
```
