# Example Apps

## Create Chipi App

<Frame>
  <img className="block dark:hidden" src="/images/create-chipi-app-light.svg" />
  <img className="hidden dark:block" src="/images/create-chipi-app-dark.svg" />
</Frame>

Whether you need React, Next.js, or Expo, we've got you covered. Simply run

```bash
npx create-chipi-app@latest
```

and follow the prompts!

<Tip>
  Our example apps come with pre-configured authentication flows, UI components, and best practices already set up. They're perfect for:
  
  * Learning how to use Chipi Pay
  * Starting a new project
  * Testing different features
</Tip>
